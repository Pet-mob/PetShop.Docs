# ✂️ Serviços e Preços

A tela de Serviços é onde você define o cardápio de serviços do seu petshop — o que você oferece, quanto cobra e quanto tempo cada serviço leva. Esses dados aparecem diretamente para os clientes no app.

## Como acessar

Menu lateral → **Configurações → Serviços e Preços**

---

## Cadastrando um serviço

1. Clique em **"+ Adicionar serviço"**
2. Preencha os dados:

| Campo               | Obrigatório | Descrição                                             |
| ------------------- | ----------- | ----------------------------------------------------- |
| **Nome do serviço** | ✅ Sim      | Ex: "Banho e Tosa", "Banho Simples", "Tosa Higiênica" |
| **Descrição**       | ❌ Não      | Texto que aparece para o cliente no app               |
| **Duração**         | ✅ Sim      | Tempo estimado para realizar o serviço                |
| **Ativo**           | ✅ Sim      | Se aparece no app ou não                              |

### Preço por porte

Você pode definir preços diferentes por porte de pet:

| Porte               | Como preencher                             |
| ------------------- | ------------------------------------------ |
| **Pequeno**         | Pets até 10kg                              |
| **Médio**           | Pets de 10 a 20kg                          |
| **Grande**          | Pets acima de 20kg                         |
| **Todos os portes** | Se o preço é único independente do tamanho |

::: tip
Se você cobra preços diferentes por porte, o app mostra automaticamente o preço correto para o porte do pet do cliente.
:::

3. Clique em **"Salvar serviço"**

---

## Lista de serviços

A lista exibe todos os serviços cadastrados:

| Coluna           | Descrição                        |
| ---------------- | -------------------------------- |
| **Serviço**      | Nome e descrição                 |
| **Duração**      | Tempo estimado                   |
| **Preço**        | Por porte (P/M/G) ou preço único |
| **Agendamentos** | Quantas vezes foi contratado     |
| **Status**       | Ativo / Inativo                  |

---

## Editando um serviço

1. Clique no serviço na lista
2. Altere os campos desejados
3. Clique em **"Salvar"**

As alterações são refletidas no app imediatamente. Agendamentos já confirmados com o preço antigo não são afetados.

---

## Ativando e desativando serviços

- **Ativo:** O serviço aparece no app para os clientes agendarem
- **Inativo:** O serviço não aparece para novos agendamentos

Use a desativação quando um serviço estiver temporariamente indisponível (ex: profissional de férias, produto em falta).

## Comportamento esperado

✅ **Duração no calendário** — A duração do serviço é usada para calcular automaticamente os horários disponíveis na agenda.

✅ **Preço exibido no app** — O cliente vê o preço do serviço ANTES de confirmar o agendamento.

✅ **Serviços em destaque** — Os serviços mais agendados aparecem em primeiro lugar na lista do app.

## Perguntas frequentes

**Posso ter um serviço com preço definido pelo cliente na hora?**
Não. O preço precisa ser definido no cadastro do serviço. Se o preço varia, defina o preço mínimo e combine o restante com o cliente pessoalmente.

**Como faço para um serviço aparecer apenas para clientes do app e não no portal?**
No momento, todos os serviços ativos aparecem tanto no app quanto no portal de agendamento.

**Posso cadastrar serviços para animais específicos (não cães e gatos)?**
Sim. Use a descrição do serviço para especificar (ex: "Banho para coelhos"). A categorização avançada por espécie está em desenvolvimento.
