# 🔗 Link de Agendamento

O Link de Agendamento é um dos recursos mais poderosos da plataforma. Com ele, você gera uma URL personalizada que pode ser compartilhada em qualquer canal — WhatsApp, Instagram, bio, cartão de visitas — e o cliente agenda sem precisar baixar o app.

## Como acessar

Menu lateral → **Configurações → Link de Agendamento**

---

## O que é e como funciona

Cada petshop recebe uma URL única no formato:

```
https://petshop.petmob.com.br/portal/{cnpj}
```

Ao acessar esse link, o cliente cai diretamente no **Portal de Agendamento** — uma página web otimizada para celular onde ele pode:

1. Ver seus serviços e preços
2. Escolher o pet e o serviço
3. Selecionar data e horário
4. Confirmar o agendamento

Tudo isso **sem precisar criar conta** ou baixar nada.

---

## Gerando e copiando seu link

1. Na tela de Link de Agendamento, seu link já está disponível
2. Clique em **"Copiar link"** para copiá-lo para a área de transferência
3. Cole onde quiser: WhatsApp, Instagram, e-mail, etc.

---

## QR Code

Além do link, você pode baixar um **QR Code** para:
- Impressão no balcão do petshop
- Cartão de visitas digital
- Banners e materiais gráficos

Para baixar: clique em **"Baixar QR Code"** → escolha o formato (PNG ou SVG).

---

## Personalizando o link

Você pode personalizar o "slug" (parte final da URL) para ficar mais fácil de lembrar:

- Padrão: `https://petshop.petmob.com.br/portal/petshop-123`
- Personalizado: `https://petshop.petmob.com.br/portal/meupetshop`

Para personalizar:
1. No campo **"Endereço personalizado"**, digite o nome desejado (sem espaços)
2. Use apenas letras minúsculas, números e hífens
3. Clique em **"Verificar disponibilidade"**
4. Se disponível, clique em **"Salvar"**

::: warning
Ao alterar o link personalizado, o link antigo para de funcionar. Se você já divulgou o link anterior, atualize em todos os canais.
:::

## Compartilhamento rápido

A tela oferece botões de compartilhamento direto para os canais mais usados:

| Canal | O que acontece |
|---|---|
| **WhatsApp** | Abre o WhatsApp Web com a mensagem pronta para enviar |
| **Instagram** | Copia o link para você colar na bio |
| **E-mail** | Abre seu cliente de e-mail com o link no corpo |

### Mensagem sugerida para WhatsApp

> *"Olá! Agora você pode agendar serviços para o seu pet diretamente pelo celular, sem precisar ligar ou ir até nossa loja. Acesse o link e escolha o melhor horário: [SEU LINK]* 🐾"

## Comportamento esperado

✅ **Link sempre ativo** — Enquanto sua conta estiver ativa, o link funciona 24 horas por dia, 7 dias por semana.

✅ **Horários em tempo real** — O portal exibe apenas horários disponíveis com base na sua agenda atual.

✅ **Agendamentos do portal** — Aparecem na sua agenda normalmente, com um ícone indicando que vieram pelo portal (🌐).

---

## Perguntas frequentes

**O portal de agendamento é diferente do app?**
Sim. O portal é uma página web simples, sem necessidade de instalação. O app oferece mais recursos (pontos de fidelidade, histórico, acompanhamento em tempo real).

**Posso desativar o portal temporariamente?**
Sim. Use o toggle **"Aceitar agendamentos pelo portal"** para ativar ou desativar.

**O cliente precisa criar conta para usar o portal?**
Não. O cliente fornece apenas nome, telefone e dados do pet. Sem cadastro obrigatório.
