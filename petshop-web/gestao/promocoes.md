# 📢 Promoções

::: info Disponível nos planos Pro e Elite
Esta funcionalidade está disponível nos planos Pro e Elite.
:::

Promoções são campanhas com preços especiais em serviços específicos por um período determinado. Diferente dos cupons (que exigem um código), as promoções aparecem automaticamente para todos os clientes no app.

## Como acessar

Menu lateral → **Gestão → Promoções**

---

## Diferença entre Promoções e Cupons

|                           | Promoções                         | Cupons                                     |
| ------------------------- | --------------------------------- | ------------------------------------------ |
| **Como o cliente acessa** | Visível automaticamente no app    | Precisa de código                          |
| **Para quem se aplica**   | Todos os clientes                 | Quem tem o código                          |
| **Uso ideal**             | Campanhas amplas, datas especiais | Ações segmentadas, fidelização             |
| **Exemplo**               | "Tosa em dobro toda terça-feira"  | "BEMVINDO20 — 20% no primeiro agendamento" |

---

## Criando uma promoção

1. Clique em **"+ Criar promoção"**
2. Preencha os dados:

| Campo                      | Obrigatório | Descrição                                    |
| -------------------------- | ----------- | -------------------------------------------- |
| **Nome da promoção**       | ✅ Sim      | Ex: "Semana do Pet", "Promoção de Inverno"   |
| **Descrição**              | ❌ Não      | Texto que aparece no app para o cliente      |
| **Serviço(s) incluído(s)** | ✅ Sim      | Quais serviços fazem parte da promoção       |
| **Tipo de desconto**       | ✅ Sim      | Percentual (%) ou valor fixo (R$)            |
| **Valor do desconto**      | ✅ Sim      |                                              |
| **Data de início**         | ✅ Sim      |                                              |
| **Data de fim**            | ✅ Sim      |                                              |
| **Dias da semana**         | ❌ Não      | Se a promoção vale apenas certos dias        |
| **Horários**               | ❌ Não      | Se a promoção vale apenas em certos horários |

3. Clique em **"Publicar promoção"**

---

## Como a promoção aparece no app

Quando um cliente acessa seu perfil no app ou faz uma busca, um badge **"PROMOÇÃO"** aparece nos serviços com desconto. O preço original fica riscado e o preço promocional aparece em destaque.

---

## Gerenciando promoções

A lista exibe todas as promoções criadas:

| Coluna           | Descrição                          |
| ---------------- | ---------------------------------- |
| **Nome**         | Título da promoção                 |
| **Serviço(s)**   | Serviços em promoção               |
| **Desconto**     | Valor ou percentual                |
| **Vigência**     | Período de início e fim            |
| **Agendamentos** | Vezes que a promoção foi utilizada |
| **Status**       | Ativa / Encerrada / Agendada       |

---

## Encerrando uma promoção antecipadamente

1. Localize a promoção ativa na lista
2. Clique em **"Encerrar promoção"**
3. Confirme

A promoção é removida do app imediatamente. Agendamentos já feitos com o preço promocional não são afetados.

---

## Comportamento esperado

✅ **Promoção agendada** — Você pode criar uma promoção com início futuro. Ela fica com status "Agendada" e é ativada automaticamente na data definida.

✅ **Encerramento automático** — Na data de término, a promoção é encerrada automaticamente sem necessidade de ação manual.

✅ **Conflito de promoções** — Não é possível ter duas promoções ativas para o mesmo serviço ao mesmo tempo. O sistema avisa sobre conflitos ao criar.
