# ⚖️ Comparar Petshops

A tela **Comparar Petshops** permite visualizar lado a lado os dados de dois petshops cadastrados na plataforma PetMob — útil para benchmarking e análise competitiva.

## Como acessar

Menu lateral → **Gestão → Comparar Petshops**

---

## Como usar

### 1. Selecionar os petshops

1. No campo **"Pet Shop A"**, digite o CNPJ do primeiro petshop
2. Confirme o nome do estabelecimento exibido abaixo
3. No campo **"Pet Shop B"**, digite o CNPJ do segundo petshop
4. Confirme o nome
5. Clique em **"Comparar"**

::: tip
Você pode comparar seu próprio petshop com um concorrente para identificar oportunidades de melhoria.
:::

---

## Critérios comparados

| Critério | Descrição |
|---|---|
| **Avaliação Média** | Nota média das avaliações dos tutores (0–5 ⭐) |
| **Nº de Avaliações** | Total de avaliações recebidas |
| **Total de Agendamentos** | Agendamentos realizados na plataforma |
| **Serviços Cadastrados** | Quantidade de serviços no cardápio |
| **Tempo Médio de Resposta** | Média de tempo para confirmar agendamentos |

O critério melhor entre os dois é destacado em verde. O pior é destacado em vermelho.

---

## Comportamento esperado

✅ **Dados públicos apenas** — A comparação usa somente dados visíveis publicamente no marketplace (avaliações, número de agendamentos, serviços). Nenhum dado financeiro ou privado é exibido.

✅ **CNPJ válido** — Se o CNPJ não estiver cadastrado na PetMob, uma mensagem de erro é exibida.

✅ **Atualização em tempo real** — Os dados são buscados no momento da comparação, sempre atualizados.

---

## Perguntas frequentes

**Posso comparar dois petshops que não são o meu?**
Sim. Você pode comparar quaisquer dois petshops cadastrados na plataforma.

**Os dados da comparação são confidenciais?**
Os dados exibidos são os mesmos disponíveis publicamente no app para qualquer usuário. Não há exposição de dados privados.

**A comparação fica salva?**
Não. A cada acesso à tela, você precisa inserir os CNPJs novamente. Para acompanhar benchmarking ao longo do tempo, utilize os Relatórios (Gestão → Relatórios).
